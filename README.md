# Armador de Horarios

Herramienta web simple para ayudar a estudiantes a seleccionar secciones
(NRC) y visualizar posibles topes de horario.

## Funcionalidades
- Búsqueda por NRC
- Selección desde listas desplegables
- Visualización gráfica del horario
- Detección automática de topes
- Sin backend (HTML, CSS, JS)

## Uso
1. Editar `courses.json` con las secciones disponibles
2. Abrir la web o usar GitHub Pages

## Tecnologías
- HTML
- CSS
- JavaScript (vanilla)
